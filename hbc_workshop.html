Doom Emacs

Un framework de configuración para GNU Emacs.

Es una historia tan antigua como el tiempo. Un vimmer testarudo, inquietante y melodramático, envidioso de las características de los editores de texto modernos, se desespera antes de sucumbir al lado oscuro. Esta es su configuración.

¡Instala Doom Emacs ya!
Ofrece features como:
  • Bonitos looks minimalistas inspirados en editores modernos.
  • Emulación vim opcional impulsada por el modo malvado.
  • Una estructura organizativa modular para separar preocupaciones en su configuración.
  • Espacios de trabajo aislados y persistentes (también sustituye a las pestañas vim).
  • Soporte para muchas herramientas, como docker, pass, ansible, terraform y más.

Doom es un marco de configuración para GNU Emacs diseñado para los veteranos de la quiebra de Emacs que quieren menos marco en sus marcos, un mínimo de estabilidad (y reproducibilidad) de su administrador de paquetes y el rendimiento de una configuración hecha a mano (o mejor). Puede ser una base para su propia configuración o un recurso para que los entusiastas de Emacs aprendan más sobre nuestro sistema operativo favorito.
